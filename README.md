# cpp-portfolio
C++ Data Structures & Software Engineering Portfolio

A collection of clean, performance-focused C++ applications developed in Xcode, demonstrating memory safety, algorithmic optimization, and stream validation.

📂 Featured Applications

1. Array Equivalency Evaluator (/array-equivalence-evaluator)

An optimized, sorting-independent lookup algorithm to verify if two dynamic arrays contain identical element frequencies.

Key Concepts: Frequency counters, nested loop mechanics, and $O(N^2)$ benchmark optimization without modifying underlying memory layouts.

2. Title-Case String Parser (/title-case-string-parser)

A memory-safe text parsing engine that sanitizes and systematically transforms terminal strings into standardized Title-Case syntax.

Key Concepts: Raw pointer-offset addressing, character-by-character validation, and safe buffer boundary enforcement.

3. Student Records Database (/student-records-system)

A structural console database utility featuring dynamic data storage and robust keyboard input sanitization.

Key Concepts: Structured data design (struct), runtime scaling via std::vector, and robust stream recovery routines using std::cin.clear() and std::cin.ignore().

🛠️ Technical Specifications

Language Standard: C++11 / C++14

Toolchain: Apple Clang / LLVM (Xcode Command Line Tools)

Platform: macOS / POSIX compliant
